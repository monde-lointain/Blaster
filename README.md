# Blaster
Online multiplayer shooter created in Unreal Engine 5.
WIP<br>
Current features implemented include a fully functioning character class with movement, firing and death animations, multiple weapon classes for projectile, linetrace and radial damage, score tracking, pre- and post-match cooldown periods.
